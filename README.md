#To-Do List App with Reminders (SwiftUI & Core Data)

This is a simple yet powerful To-Do List application built using SwiftUI and Core Data for iOS devices. The app allows users to create, edit, and delete tasks with full CRUD (Create, Read, Update, Delete) functionality. Additionally, it supports setting reminders for tasks using local device notifications.

#Features

Add Tasks: Create new to-do tasks with a title and optional reminder date/time.

Edit Tasks: Update task details and modify reminders.

Delete Tasks: Remove tasks you no longer need.

Core Data Integration: Persistent storage of tasks ensures your data is saved between app launches.

#Technologies Used

SwiftUI for UI development

Core Data for local persistence

UserNotifications framework for scheduling local notifications

#How to Use

Clone or download the repository.

Open the Xcode project.

Run the app on a simulator or physical iOS device.

Add, edit, or delete tasks as needed.

Set reminders to receive notifications at the specified times.
